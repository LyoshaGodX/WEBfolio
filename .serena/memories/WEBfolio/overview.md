Project: WEBfolio — веб-портфолио по направлению «Информатика и вычислительная техника». 

Краткое назначение:
- Статический сайт-портфолио, отображающий дисциплины, семестры и автоматически сгенерированные теги навыков.

Техстек:
- Язык: Python
- Статический генератор: Pelican (pelican==4.10.2)
- Шаблоны: Jinja2
- Формат контента: Markdown (файлы в `content/`, посты в `content/posts`)
- Плагины: `custom_filters` (локальный), кастомная тема в `theme/mytheme`

Ключевые скрипты и файлы:
- `update_skills_tags.py` — обновляет переменную `SKILLS_TAGS` в `pelicanconf.py` (парсит `.md` в `content/posts`)
- `generate_skills_tags.py` — ручной парсер для просмотра
- `pelicanconf.py`, `publishconf.py` — конфигурация Pelican
- `Makefile` — удобные команды для сборки и сервера (см. `make html`, `make serve`, `make update-tags`)
- `requirements.txt` — список зависимостей

Структура проекта (основные пути):
- `content/` — контент (посты, страницы)
- `theme/` — тема сайта (`theme/mytheme`)
- `output/` — сгенерированный сайт (генерируется)

Отсутствует/нужна проверка:
- Нет явных конфигураций для линтинга/форматирования (например, `pyproject.toml`, `.flake8`, `black`), нужно уточнить предпочтения по стилю кода.
- Нет тестов автоматических; CI в `.github/` можно проверить при желании.

Окружение: проект разворачивается на Windows (локальная разработка указана как Windows в onboarding).
